# @noonnu/okgung

읏궁체 - 독특한 글씨체가 많다

![example](./example.png)

## Install

```bash
npm install @noonnu/okgung --save
```

### Import the CSS file

```js
import '@noonnu/okgung' // esm
// or
require('@noonnu/okgung') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/okgung');
```

## Usage

```css
body {
    font-family: OKGUNG;
}
```
