# @noonnu/one-mobile-title
원스토어 모바일고딕 제목체 - 고양이를 보면 기분이 좋다

## Install
```sh
npm i @noonnu/one-mobile-title
```
## Usage
### cjs
```js
require("@noonnu/one-mobile-title")
```
### esm
```js
import "@noonnu/one-mobile-title"
```
### css-loader
```css
@import "~@noonnu/one-mobile-title/index.css"
```
