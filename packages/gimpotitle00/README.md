## Install
```sh
npm i @noonnu/gimpotitle00
```
## Usage
### cjs
```js
require("@noonnu/gimpotitle00")
```
### esm
```js
import "@noonnu/gimpotitle00"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/gimpotitle00/index.css"
```
