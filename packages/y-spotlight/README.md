# @noonnu/y-spotlight
Y 너만을 비춤체 - 귀여운 이모티콘도 있는 폰트다 왇 왘

## Install
```sh
npm i @noonnu/y-spotlight
```
## Usage
### cjs
```js
require("@noonnu/y-spotlight")
```
### esm
```js
import "@noonnu/y-spotlight"
```
### css-loader
```css
@import "~@noonnu/y-spotlight/index.css"
```
