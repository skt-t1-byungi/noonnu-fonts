# @noonnu/kalguksu
칼국수 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/kalguksu
```
## Usage
### cjs
```js
require("@noonnu/kalguksu")
```
### esm
```js
import "@noonnu/kalguksu"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/kalguksu/index.css"
```
