# @noonnu/cookierunotf-bold
쿠키런 Bold - 쿠키 먹으면서 하는 게임

## Install
```sh
npm i @noonnu/cookierunotf-bold
```
## Usage
### cjs
```js
require("@noonnu/cookierunotf-bold")
```
### esm
```js
import "@noonnu/cookierunotf-bold"
```
### css-loader
```css
@import "~@noonnu/cookierunotf-bold/index.css"
```
