# @noonnu/mini-handwriting
미니 손글씨 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/mini-handwriting
```
## Usage
### cjs
```js
require("@noonnu/mini-handwriting")
```
### esm
```js
import "@noonnu/mini-handwriting"
```
### css-loader
```css
@import "~@noonnu/mini-handwriting/index.css"
```
