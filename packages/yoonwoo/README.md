# @noonnu/yoonwoo

온글잎 윤우체 - 날씨 엄청 좋을때 바닷가 놀러가고 싶다

![example](./example.png)

## Install

```bash
npm install @noonnu/yoonwoo --save
```

### Import the CSS file

```js
import '@noonnu/yoonwoo' // esm
// or
require('@noonnu/yoonwoo') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/yoonwoo');
```

## Usage

```css
body {
    font-family: Yoonwoo;
}
```
