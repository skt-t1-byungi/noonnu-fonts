# @noonnu/goyangilsan
고양일산체 - 일산에는 킨텍스가 있다

## Install
```sh
npm i @noonnu/goyangilsan
```
## Usage
### cjs
```js
require("@noonnu/goyangilsan")
```
### esm
```js
import "@noonnu/goyangilsan"
```
### css-loader
```css
@import "~@noonnu/goyangilsan/index.css"
```
