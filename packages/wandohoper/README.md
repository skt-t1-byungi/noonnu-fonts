# @noonnu/wandohoper
완도희망체 Regular - 청정하고 넉넉한 바다를 닮은 서체

## Install
```sh
npm i @noonnu/wandohoper
```
## Usage
### cjs
```js
require("@noonnu/wandohoper")
```
### esm
```js
import "@noonnu/wandohoper"
```
### css-loader
```css
@import "~@noonnu/wandohoper/index.css"
```
