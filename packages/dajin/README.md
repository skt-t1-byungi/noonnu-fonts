# @noonnu/dajin
다진체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/dajin
```
## Import
### CJS
```js
require("@noonnu/dajin")
```
### ESM
```js
import "@noonnu/dajin"
```
### CSS 
```css
@import "~@noonnu/dajin/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Dajin", serif;
}
```
