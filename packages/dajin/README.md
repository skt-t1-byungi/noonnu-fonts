# @noonnu/dajin

다진체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

![example](./example.png)

## Install

```bash
npm install @noonnu/dajin --save
```

### Import the CSS file

```js
import '@noonnu/dajin' // esm
// or
require('@noonnu/dajin') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/dajin');
```

## Usage

```css
body {
    font-family: Dajin;
}
```
