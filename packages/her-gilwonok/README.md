```sh
npm i @noonnu/her-gilwonok
```

```js
require("@noonnu/her-gilwonok")
```

### esm
```js
import "@noonnu/her-gilwonok"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/her-gilwonok/index.css"
```
