# @noonnu/bisang

비상체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

![example](./example.png)

## Install

```bash
npm install @noonnu/bisang --save
```

### Import the CSS file

```js
import '@noonnu/bisang' // esm
// or
require('@noonnu/bisang') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/bisang');
```

## Usage

```css
body {
    font-family: Bisang;
}
```
