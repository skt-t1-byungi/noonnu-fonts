# @noonnu/bisang
비상체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/bisang
```
## Import
### CJS
```js
require("@noonnu/bisang")
```
### ESM
```js
import "@noonnu/bisang"
```
### CSS 
```css
@import "~@noonnu/bisang/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Bisang", serif;
}
```
