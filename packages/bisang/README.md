# @noonnu/bisang
비상체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/bisang
```
## Usage
### cjs
```js
require("@noonnu/bisang")
```
### esm
```js
import "@noonnu/bisang"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/bisang/index.css"
```
