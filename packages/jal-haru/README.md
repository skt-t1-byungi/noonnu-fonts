# @noonnu/jal-haru
잘풀리는하루체 - 이 폰트를 사용하면 왠지 하루가 잘 풀릴 것 같다

## Install
```sh
npm i @noonnu/jal-haru
```
## Usage
### cjs
```js
require("@noonnu/jal-haru")
```
### esm
```js
import "@noonnu/jal-haru"
```
### css-loader
```css
@import "~@noonnu/jal-haru/index.css"
```
