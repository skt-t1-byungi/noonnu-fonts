# @noonnu/kimnamyun
김남윤체 - 변화는 당신으로부터 이루어져요

## Install
```sh
npm i @noonnu/kimnamyun
```
## Usage
### cjs
```js
require("@noonnu/kimnamyun")
```
### esm
```js
import "@noonnu/kimnamyun"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/kimnamyun/index.css"
```
