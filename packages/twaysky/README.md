# @noonnu/twaysky

티웨이하늘체 - 제주도 바다가 왠지 더 예쁜 듯

![example](./example.png)

## Install

```bash
npm install @noonnu/twaysky --save
```

### Import the CSS file

```js
import '@noonnu/twaysky' // esm
// or
require('@noonnu/twaysky') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/twaysky');
```

## Usage

```css
body {
    font-family: twaysky;
}
```
