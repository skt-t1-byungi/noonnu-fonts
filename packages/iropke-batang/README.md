이롭게바탕체 - 가볍고 미려한 이롭게 바탕체

## Install
```sh
npm i @noonnu/iropke-batang
```
## Usage
### cjs
```js
require("@noonnu/iropke-batang")
```
### esm
```js
import "@noonnu/iropke-batang"
```
### css-loader
```css
@import "~@noonnu/iropke-batang/index.css"
```
