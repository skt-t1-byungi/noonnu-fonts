# @noonnu/yangjinche

양진체 - 레트로하고 키치한 느낌의 서체

![example](./example.png)

## Install

```bash
npm install @noonnu/yangjinche --save
```

### Import the CSS file

```js
import '@noonnu/yangjinche' // esm
// or
require('@noonnu/yangjinche') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/yangjinche');
```

## Usage

```css
body {
    font-family: 양진체;
}
```
