# @noonnu/pretendard-regular
프리텐다드 - 세상에 이런 폰트가 나오다니 천재인듯

## Install
```sh
npm i @noonnu/pretendard-regular
```
## Usage
### cjs
```js
require("@noonnu/pretendard-regular")
```
### esm
```js
import "@noonnu/pretendard-regular"
```
### css-loader
```css
@import "~@noonnu/pretendard-regular/index.css"
```
