# @noonnu/ainmom
아인맘 손글씨 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/ainmom
```
## Import
### CJS
```js
require("@noonnu/ainmom")
```
### ESM
```js
import "@noonnu/ainmom"
```
### CSS 
```css
@import "~@noonnu/ainmom/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Ainmom", serif;
}
```
