# @noonnu/yoon-sah

Yoon 아혜 - 아무것도 하기 싫은 날

![example](./example.png)

## Install

```bash
npm install @noonnu/yoon-sah --save
```

### Import the CSS file

```js
import '@noonnu/yoon-sah' // esm
// or
require('@noonnu/yoon-sah') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/yoon-sah');
```

## Usage

```css
body {
    font-family: YoonSAH;
}
```
