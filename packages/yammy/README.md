# @noonnu/yammy
맛있는체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/yammy
```
## Import
### CJS
```js
require("@noonnu/yammy")
```
### ESM
```js
import "@noonnu/yammy"
```
### CSS 
```css
@import "~@noonnu/yammy/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Yammy", serif;
}
```
