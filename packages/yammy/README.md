# @noonnu/yammy

맛있는체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

![example](./example.png)

## Install

```bash
npm install @noonnu/yammy --save
```

### Import the CSS file

```js
import '@noonnu/yammy' // esm
// or
require('@noonnu/yammy') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/yammy');
```

## Usage

```css
body {
    font-family: Yammy;
}
```
