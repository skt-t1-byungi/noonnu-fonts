# @noonnu/daehan
대한체 - 인생 한 번 살지 두 번 살겠니

## Install
```sh
npm i @noonnu/daehan
```
## Import
### CJS
```js
require("@noonnu/daehan")
```
### ESM
```js
import "@noonnu/daehan"
```
### CSS 
```css
@import "~@noonnu/daehan/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Daehan", serif;
}
```
