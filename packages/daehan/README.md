# @noonnu/daehan
대한체 - 인생 한 번 살지 두 번 살겠니

## Install
```sh
npm i @noonnu/daehan
```
## Usage
### cjs
```js
require("@noonnu/daehan")
```
### esm
```js
import "@noonnu/daehan"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/daehan/index.css"
```
