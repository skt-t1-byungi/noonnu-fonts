## Install
```sh
npm i @noonnu/bbtreecr
```
## Usage
### cjs
```js
require("@noonnu/bbtreecr")
```
### esm
```js
import "@noonnu/bbtreecr"
```
### css-loader
```css
@import "~@noonnu/bbtreecr/index.css"
```
