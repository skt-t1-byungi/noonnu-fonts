# @noonnu/napjakblock
가비아 납작블럭체 - 꾸준하면 무엇이든 할 수 있어

## Install
```sh
npm i @noonnu/napjakblock
```
## Usage
### cjs
```js
require("@noonnu/napjakblock")
```
### esm
```js
import "@noonnu/napjakblock"
```
### css-loader
```css
@import "~@noonnu/napjakblock/index.css"
```
