# @noonnu/nexon-lv2-gothic
넥슨 Lv.2 고딕 Regular - 올려야하는 폰트가 너무너무 많다..

## Install
```sh
npm i @noonnu/nexon-lv2-gothic
```
## Usage
### cjs
```js
require("@noonnu/nexon-lv2-gothic")
```
### esm
```js
import "@noonnu/nexon-lv2-gothic"
```
### css-loader
```css
@import "~@noonnu/nexon-lv2-gothic/index.css"
```
