# @noonnu/chilgok-cye
칠곡할매 추유을체 - 아기자기 귀여운 할머니의 글씨

## Install
```sh
npm i @noonnu/chilgok-cye
```
## Usage
### cjs
```js
require("@noonnu/chilgok-cye")
```
### esm
```js
import "@noonnu/chilgok-cye"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/chilgok-cye/index.css"
```
