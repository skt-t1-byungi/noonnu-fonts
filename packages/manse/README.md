# @noonnu/manse
만세체 - 태극기를 들고 만세를 외친다

## Install
```sh
npm i @noonnu/manse
```
## Import
### CJS
```js
require("@noonnu/manse")
```
### ESM
```js
import "@noonnu/manse"
```
### CSS 
```css
@import "~@noonnu/manse/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Manse", serif;
}
```
