# @noonnu/haram
하람체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/haram
```
## Import
### CJS
```js
require("@noonnu/haram")
```
### ESM
```js
import "@noonnu/haram"
```
### CSS 
```css
@import "~@noonnu/haram/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Haram", serif;
}
```
