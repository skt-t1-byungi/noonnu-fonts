# @noonnu/haram

하람체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

![example](./example.png)

## Install

```bash
npm install @noonnu/haram --save
```

### Import the CSS file

```js
import '@noonnu/haram' // esm
// or
require('@noonnu/haram') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/haram');
```

## Usage

```css
body {
    font-family: Haram;
}
```
