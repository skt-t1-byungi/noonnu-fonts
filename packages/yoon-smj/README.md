# @noonnu/yoon-smj

Yoon 민준 - 손글씨 폰트가 무척 많다

![example](./example.png)

## Install

```bash
npm install @noonnu/yoon-smj --save
```

### Import the CSS file

```js
import '@noonnu/yoon-smj' // esm
// or
require('@noonnu/yoon-smj') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/yoon-smj');
```

## Usage

```css
body {
    font-family: YoonSMJ;
}
```
