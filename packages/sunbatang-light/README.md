# @noonnu/sunbatang-light
순바탕Light - 바람이 산들산들 불고 하늘이 파란 날

## Install
```sh
npm i @noonnu/sunbatang-light
```
## Usage
### cjs
```js
require("@noonnu/sunbatang-light")
```
### esm
```js
import "@noonnu/sunbatang-light"
```
### css-loader
```css
@import "~@noonnu/sunbatang-light/index.css"
```
