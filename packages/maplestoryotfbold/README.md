# @noonnu/maplestoryotfbold
넥슨 메이플스토리 Bold - 동글동글 귀여운 메이플스토리체

## Install
```sh
npm i @noonnu/maplestoryotfbold
```
## Usage
### cjs
```js
require("@noonnu/maplestoryotfbold")
```
### esm
```js
import "@noonnu/maplestoryotfbold"
```
### css-loader
```css
@import "~@noonnu/maplestoryotfbold/index.css"
```
