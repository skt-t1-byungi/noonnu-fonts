# @noonnu/maruburi-regular
마루 부리 - 명조체의 돌기인 마루 부리

## Install
```sh
npm i @noonnu/maruburi-regular
```
## Usage
### cjs
```js
require("@noonnu/maruburi-regular")
```
### esm
```js
import "@noonnu/maruburi-regular"
```
### css-loader
```css
@import "~@noonnu/maruburi-regular/index.css"
```
