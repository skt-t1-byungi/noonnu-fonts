# @noonnu/leehyunji
이숲체 - 지구가 둥글다는건 아무도 믿지 않았어

## Install
```sh
npm i @noonnu/leehyunji
```
## Usage
### cjs
```js
require("@noonnu/leehyunji")
```
### esm
```js
import "@noonnu/leehyunji"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/leehyunji/index.css"
```
