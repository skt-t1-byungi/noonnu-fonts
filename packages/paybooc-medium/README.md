# @noonnu/paybooc-medium
페이북글꼴 Medium - 페이북으로 결제하면 할인해주는거 좋다

## Install
```sh
npm i @noonnu/paybooc-medium
```
## Usage
### cjs
```js
require("@noonnu/paybooc-medium")
```
### esm
```js
import "@noonnu/paybooc-medium"
```
### css-loader
```css
@import "~@noonnu/paybooc-medium/index.css"
```
