# @noonnu/kotra-gothic
코트라 고딕체 - 가을은 왜 이렇게 짧은거야

## Install
```sh
npm i @noonnu/kotra-gothic
```
## Usage
### cjs
```js
require("@noonnu/kotra-gothic")
```
### esm
```js
import "@noonnu/kotra-gothic"
```
### css-loader
```css
@import "~@noonnu/kotra-gothic/index.css"
```
