# @noonnu/openas
유토이미지 - 잠을 많이 잤더니 기분이 좋다

## Install
```sh
npm i @noonnu/openas
```
## Import
### CJS
```js
require("@noonnu/openas")
```
### ESM
```js
import "@noonnu/openas"
```
### CSS 
```css
@import "~@noonnu/openas/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Openas", serif;
}
```
