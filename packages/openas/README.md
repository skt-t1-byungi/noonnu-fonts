# @noonnu/openas

유토이미지 - 잠을 많이 잤더니 기분이 좋다

![example](./example.png)

## Install

```bash
npm install @noonnu/openas --save
```

### Import the CSS file

```js
import '@noonnu/openas' // esm
// or
require('@noonnu/openas') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/openas');
```

## Usage

```css
body {
    font-family: Openas;
}
```
