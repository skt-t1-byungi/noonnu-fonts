# @noonnu/openas
유토이미지 - 잠을 많이 잤더니 기분이 좋다

## Install
```sh
npm i @noonnu/openas
```
## Usage
### cjs
```js
require("@noonnu/openas")
```
### esm
```js
import "@noonnu/openas"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/openas/index.css"
```
