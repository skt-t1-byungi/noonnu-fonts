# @noonnu/hanbatangb
함초롬바탕체 Bold - 해외여행을 간게 언제였는지 기억도 안난다

## Install
```sh
npm i @noonnu/hanbatangb
```
## Usage
### cjs
```js
require("@noonnu/hanbatangb")
```
### esm
```js
import "@noonnu/hanbatangb"
```
### css-loader
```css
@import "~@noonnu/hanbatangb/index.css"
```
