# @noonnu/mapo-hongdae-freedom

마포홍대프리덤 - 홍대에는 사람이 너무 많아

![example](./example.png)

## Install

```bash
npm install @noonnu/mapo-hongdae-freedom --save
```

### Import the CSS file

```js
import '@noonnu/mapo-hongdae-freedom' // esm
// or
require('@noonnu/mapo-hongdae-freedom') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/mapo-hongdae-freedom');
```

## Usage

```css
body {
    font-family: MapoHongdaeFreedom;
}
```
