# @noonnu/gilbeottg
길벗체 TG - 겨울은 역시 호떡이지

## Install
```sh
npm i @noonnu/gilbeottg
```
## Import
### CJS
```js
require("@noonnu/gilbeottg")
```
### ESM
```js
import "@noonnu/gilbeottg"
```
### CSS 
```css
@import "~@noonnu/gilbeottg/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "GilbeotTG", serif;
}
```
