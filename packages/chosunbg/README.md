# @noonnu/chosunbg
조선견고딕 - 견고해보이는 견고딕

## Install
```sh
npm i @noonnu/chosunbg
```
## Import
### CJS
```js
require("@noonnu/chosunbg")
```
### ESM
```js
import "@noonnu/chosunbg"
```
### CSS 
```css
@import "~@noonnu/chosunbg/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "ChosunBg", serif;
}
```
