```sh
npm i @noonnu/binggraesamanco
```

```js
require("@noonnu/binggraesamanco")
```
### esm
```js
import "@noonnu/binggraesamanco"
```
### css-loader
```css
@import "~@noonnu/binggraesamanco/index.css"
```
