# @noonnu/daehan-bold
대한체 Bold - 대한민국독립만세 눈누에 업데이트!

## Install
```sh
npm i @noonnu/daehan-bold
```
## Usage
### cjs
```js
require("@noonnu/daehan-bold")
```
### esm
```js
import "@noonnu/daehan-bold"
```
### css-loader
```css
@import "~@noonnu/daehan-bold/index.css"
```
