# @noonnu/hanbatang
함초롬바탕체 Regular - 피크닉하기 정말 좋은 날씨다

## Install
```sh
npm i @noonnu/hanbatang
```
## Usage
### cjs
```js
require("@noonnu/hanbatang")
```
### esm
```js
import "@noonnu/hanbatang"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/hanbatang/index.css"
```
