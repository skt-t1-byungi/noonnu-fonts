# @noonnu/galmuri11

갈무리11 - 졸린데 너무 할일이 많아

![example](./example.png)

## Install

```bash
npm install @noonnu/galmuri11 --save
```

### Import the CSS file

```js
import '@noonnu/galmuri11' // esm
// or
require('@noonnu/galmuri11') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/galmuri11');
```

## Usage

```css
body {
    font-family: Galmuri11;
}
```
