# @noonnu/kanibuk

간이벽온방체 - 사람이 한치 앞을 못본다니까

![example](./example.png)

## Install

```bash
npm install @noonnu/kanibuk --save
```

### Import the CSS file

```js
import '@noonnu/kanibuk' // esm
// or
require('@noonnu/kanibuk') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/kanibuk');
```

## Usage

```css
body {
    font-family: Kanibuk;
}
```
