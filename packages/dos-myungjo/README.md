# @noonnu/dos-myungjo

도스명조 - 겨울이라서 살이 찌는 걸까

![example](./example.png)

## Install

```bash
npm install @noonnu/dos-myungjo --save
```

### Import the CSS file

```js
import '@noonnu/dos-myungjo' // esm
// or
require('@noonnu/dos-myungjo') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/dos-myungjo');
```

## Usage

```css
body {
    font-family: DOSMyungjo;
}
```
