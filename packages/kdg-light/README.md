# @noonnu/kdg-light
솔뫼 김대건체 L - 조선 최초의 가톨릭사제 김대건 신부

## Install
```sh
npm i @noonnu/kdg-light
```
## Usage
### cjs
```js
require("@noonnu/kdg-light")
```
### esm
```js
import "@noonnu/kdg-light"
```
### css-loader
```css
@import "~@noonnu/kdg-light/index.css"
```
