# @noonnu/one-mobile-regular
원스토어 모바일고딕 본문체 - 따뜻한 밀크티를 마시고 자자

## Install
```sh
npm i @noonnu/one-mobile-regular
```
## Usage
### cjs
```js
require("@noonnu/one-mobile-regular")
```
### esm
```js
import "@noonnu/one-mobile-regular"
```
### css-loader
```css
@import "~@noonnu/one-mobile-regular/index.css"
```
