# @noonnu/eoe-zno_l
남양주고딕L - 남양주에 놀러가고 싶다

## Install
```sh
npm i @noonnu/eoe-zno_l
```
## Import
### CJS
```js
require("@noonnu/eoe-zno_l")
```
### ESM
```js
import "@noonnu/eoe-zno_l"
```
### CSS 
```css
@import "~@noonnu/eoe-zno_l/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Eoe_Zno_L", serif;
}
```
