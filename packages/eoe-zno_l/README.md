# @noonnu/eoe-zno_l
남양주고딕L - 남양주에 놀러가고 싶다

## Install
```sh
npm i @noonnu/eoe-zno_l
```
## Usage
### cjs
```js
require("@noonnu/eoe-zno_l")
```
### esm
```js
import "@noonnu/eoe-zno_l"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/eoe-zno_l/index.css"
```
