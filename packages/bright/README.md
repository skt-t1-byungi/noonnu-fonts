# @noonnu/bright
밝은체 - 어제의 나를 때리고 싶다

## Install
```sh
npm i @noonnu/bright
```
## Usage
### cjs
```js
require("@noonnu/bright")
```
### esm
```js
import "@noonnu/bright"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/bright/index.css"
```
