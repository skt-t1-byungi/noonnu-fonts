# @noonnu/bright
밝은체 - 어제의 나를 때리고 싶다

## Install
```sh
npm i @noonnu/bright
```
## Import
### CJS
```js
require("@noonnu/bright")
```
### ESM
```js
import "@noonnu/bright"
```
### CSS 
```css
@import "~@noonnu/bright/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Bright", serif;
}
```
