# @noonnu/bright

밝은체 - 어제의 나를 때리고 싶다

![example](./example.png)

## Install

```bash
npm install @noonnu/bright --save
```

### Import the CSS file

```js
import '@noonnu/bright' // esm
// or
require('@noonnu/bright') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/bright');
```

## Usage

```css
body {
    font-family: Bright;
}
```
