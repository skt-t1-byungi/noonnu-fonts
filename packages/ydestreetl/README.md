# @noonnu/ydestreetl
Y이드스트릿체 Light - 와아아아아이드한 폰트

## Install
```sh
npm i @noonnu/ydestreetl
```
## Usage
### cjs
```js
require("@noonnu/ydestreetl")
```
### esm
```js
import "@noonnu/ydestreetl"
```
### css-loader
```css
@import "~@noonnu/ydestreetl/index.css"
```
