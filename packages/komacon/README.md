# @noonnu/komacon

만화진흥원체 - 웹툰보다가 밤샌적 있는 사람 손

![example](./example.png)

## Install

```bash
npm install @noonnu/komacon --save
```

### Import the CSS file

```js
import '@noonnu/komacon' // esm
// or
require('@noonnu/komacon') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/komacon');
```

## Usage

```css
body {
    font-family: KOMACON;
}
```
