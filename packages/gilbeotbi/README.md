# @noonnu/gilbeotbi
길벗체 BI - 주말에 회사일 하지말자

## Install
```sh
npm i @noonnu/gilbeotbi
```
## Import
### CJS
```js
require("@noonnu/gilbeotbi")
```
### ESM
```js
import "@noonnu/gilbeotbi"
```
### CSS 
```css
@import "~@noonnu/gilbeotbi/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "GilbeotBI", serif;
}
```
