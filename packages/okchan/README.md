# @noonnu/okchan

읏찬체 - 물을 많이 마셔야하는데

![example](./example.png)

## Install

```bash
npm install @noonnu/okchan --save
```

### Import the CSS file

```js
import '@noonnu/okchan' // esm
// or
require('@noonnu/okchan') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/okchan');
```

## Usage

```css
body {
    font-family: OKCHAN;
}
```
