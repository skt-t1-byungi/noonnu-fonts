# @noonnu/kotra-bold-bold
코트라 볼드체 - 가을이 너무 금방 추워져버렸다

## Install
```sh
npm i @noonnu/kotra-bold-bold
```
## Usage
### cjs
```js
require("@noonnu/kotra-bold-bold")
```
### esm
```js
import "@noonnu/kotra-bold-bold"
```
### css-loader
```css
@import "~@noonnu/kotra-bold-bold/index.css"
```
