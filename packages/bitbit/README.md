# @noonnu/bitbit

던파 비트비트체 - 붕어빵의 계절이 돌아왔다

![example](./example.png)

## Install

```bash
npm install @noonnu/bitbit --save
```

### Import the CSS file

```js
import '@noonnu/bitbit' // esm
// or
require('@noonnu/bitbit') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/bitbit');
```

## Usage

```css
body {
    font-family: bitbit;
}
```
