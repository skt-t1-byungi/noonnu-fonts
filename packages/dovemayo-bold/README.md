# @noonnu/dovemayo-bold
둘기마요Bold - 비둘기는 구구구 나는 구구콘 먹고 싶다

## Install
```sh
npm i @noonnu/dovemayo-bold
```
## Usage
### cjs
```js
require("@noonnu/dovemayo-bold")
```
### esm
```js
import "@noonnu/dovemayo-bold"
```
### css-loader
```css
@import "~@noonnu/dovemayo-bold/index.css"
```
