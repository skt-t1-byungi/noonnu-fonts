# @noonnu/hana-handwriting
하나손글씨 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/hana-handwriting
```
## Usage
### cjs
```js
require("@noonnu/hana-handwriting")
```
### esm
```js
import "@noonnu/hana-handwriting"
```
### css-loader
```css
@import "~@noonnu/hana-handwriting/index.css"
```
