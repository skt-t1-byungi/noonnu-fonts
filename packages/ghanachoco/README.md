# @noonnu/ghanachoco
가나초콜릿체 - 부드러움 한 조각, 가나 초콜릿

## Install
```sh
npm i @noonnu/ghanachoco
```
## Usage
### cjs
```js
require("@noonnu/ghanachoco")
```
### esm
```js
import "@noonnu/ghanachoco"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/ghanachoco/index.css"
```
