# @noonnu/kimjuim

야근하는 김주인 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

![example](./example.png)

## Install

```bash
npm install @noonnu/kimjuim --save
```

### Import the CSS file

```js
import '@noonnu/kimjuim' // esm
// or
require('@noonnu/kimjuim') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/kimjuim');
```

## Usage

```css
body {
    font-family: Kimjuim;
}
```
