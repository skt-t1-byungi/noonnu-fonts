# @noonnu/chosunlo
조선로고체 - 이걸로 조선일보쓰면 조선일보 로고

## Install
```sh
npm i @noonnu/chosunlo
```
## Usage
### cjs
```js
require("@noonnu/chosunlo")
```
### esm
```js
import "@noonnu/chosunlo"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/chosunlo/index.css"
```
