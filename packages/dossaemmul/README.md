# @noonnu/dossaemmul
도스샘물 - 다람쥐 헌 쳇바퀴에 데굴데굴

## Install
```sh
npm i @noonnu/dossaemmul
```
## Usage
### cjs
```js
require("@noonnu/dossaemmul")
```
### esm
```js
import "@noonnu/dossaemmul"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/dossaemmul/index.css"
```
