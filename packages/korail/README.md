# @noonnu/korail

코레일체 - 어떻게 전철없이 살겠어

![example](./example.png)

## Install

```bash
npm install @noonnu/korail --save
```

### Import the CSS file

```js
import '@noonnu/korail' // esm
// or
require('@noonnu/korail') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/korail');
```

## Usage

```css
body {
    font-family: KORAIL;
}
```
