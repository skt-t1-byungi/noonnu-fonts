# @noonnu/777flower
플라워 - 요즘 꽃집이 많이 보이는 것 같아

## Install
```sh
npm i @noonnu/777flower
```
## Usage
### cjs
```js
require("@noonnu/777flower")
```
### esm
```js
import "@noonnu/777flower"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/777flower/index.css"
```
