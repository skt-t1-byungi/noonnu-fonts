# @noonnu/dokdo
대한민국 독도체 - 대한민국의 영토는 한반도와 그 부속도서로 한다

## Install
```sh
npm i @noonnu/dokdo
```
## Import
### CJS
```js
require("@noonnu/dokdo")
```
### ESM
```js
import "@noonnu/dokdo"
```
### CSS 
```css
@import "~@noonnu/dokdo/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Dokdo", serif;
}
```
