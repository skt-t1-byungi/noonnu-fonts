# @noonnu/youth
청소년체 - 꽃씨는 흙을 뚫고 얼음을 뚫고

## Install
```sh
npm i @noonnu/youth
```
## Import
### CJS
```js
require("@noonnu/youth")
```
### ESM
```js
import "@noonnu/youth"
```
### CSS 
```css
@import "~@noonnu/youth/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Youth", serif;
}
```
