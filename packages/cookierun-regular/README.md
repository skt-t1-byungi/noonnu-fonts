# @noonnu/cookierun-regular
쿠키런 Regular - 오 쿠키런 폰트 Regular도 나왔다

## Install
```sh
npm i @noonnu/cookierun-regular
```
## Usage
### cjs
```js
require("@noonnu/cookierun-regular")
```
### esm
```js
import "@noonnu/cookierun-regular"
```
### css-loader
```css
@import "~@noonnu/cookierun-regular/index.css"
```
