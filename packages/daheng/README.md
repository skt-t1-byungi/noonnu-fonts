# @noonnu/daheng
다행체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/daheng
```
## Import
### CJS
```js
require("@noonnu/daheng")
```
### ESM
```js
import "@noonnu/daheng"
```
### CSS 
```css
@import "~@noonnu/daheng/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Daheng", serif;
}
```
