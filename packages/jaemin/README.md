# @noonnu/jaemin
한글재민체 - 입에 단 건 몸엔 안 좋지만 정신건강엔 좋다

## Install
```sh
npm i @noonnu/jaemin
```
## Import
### CJS
```js
require("@noonnu/jaemin")
```
### ESM
```js
import "@noonnu/jaemin"
```
### CSS 
```css
@import "~@noonnu/jaemin/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Jaemin", serif;
}
```
