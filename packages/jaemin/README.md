# @noonnu/jaemin
한글재민체 - 입에 단 건 몸엔 안 좋지만 정신건강엔 좋다

## Install
```sh
npm i @noonnu/jaemin
```
## Usage
### cjs
```js
require("@noonnu/jaemin")
```
### esm
```js
import "@noonnu/jaemin"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/jaemin/index.css"
```
