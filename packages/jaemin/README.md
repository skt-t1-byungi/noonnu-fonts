# @noonnu/jaemin

한글재민체 - 입에 단 건 몸엔 안 좋지만 정신건강엔 좋다

![example](./example.png)

## Install

```bash
npm install @noonnu/jaemin --save
```

### Import the CSS file

```js
import '@noonnu/jaemin' // esm
// or
require('@noonnu/jaemin') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/jaemin');
```

## Usage

```css
body {
    font-family: Jaemin;
}
```
