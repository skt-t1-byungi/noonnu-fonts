# @noonnu/miwon

미원체 - 감칠맛의 비법은 미원

![example](./example.png)

## Install

```bash
npm install @noonnu/miwon --save
```

### Import the CSS file

```js
import '@noonnu/miwon' // esm
// or
require('@noonnu/miwon') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/miwon');
```

## Usage

```css
body {
    font-family: Miwon;
}
```
