# @noonnu/foureyes
안경잡이체 - 별을 보고 걸어가는 사람이 되라

## Install
```sh
npm i @noonnu/foureyes
```
## Usage
### cjs
```js
require("@noonnu/foureyes")
```
### esm
```js
import "@noonnu/foureyes"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/foureyes/index.css"
```
