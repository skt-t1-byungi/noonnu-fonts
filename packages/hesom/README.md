# @noonnu/hesom

온글잎 해솜체 - 온글잎이 만든 무료 손글씨 폰트

![example](./example.png)

## Install

```bash
npm install @noonnu/hesom --save
```

### Import the CSS file

```js
import '@noonnu/hesom' // esm
// or
require('@noonnu/hesom') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/hesom');
```

## Usage

```css
body {
    font-family: Hesom;
}
```
