# @noonnu/hesom
온글잎 해솜체 - 온글잎이 만든 무료 손글씨 폰트

## Install
```sh
npm i @noonnu/hesom
```
## Usage
### cjs
```js
require("@noonnu/hesom")
```
### esm
```js
import "@noonnu/hesom"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/hesom/index.css"
```
