# @noonnu/eoe-ye_m
남양주다산M - 다산 정약용 선생의 얼을 담은 폰트

## Install
```sh
npm i @noonnu/eoe-ye_m
```
## Usage
### cjs
```js
require("@noonnu/eoe-ye_m")
```
### esm
```js
import "@noonnu/eoe-ye_m"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/eoe-ye_m/index.css"
```
