# @noonnu/hahmlet-semibold
함렛 SemiBold - 웨이트 진짜 다양하다

## Install
```sh
npm i @noonnu/hahmlet-semibold
```
## Usage
### cjs
```js
require("@noonnu/hahmlet-semibold")
```
### esm
```js
import "@noonnu/hahmlet-semibold"
```
### css-loader
```css
@import "~@noonnu/hahmlet-semibold/index.css"
```
