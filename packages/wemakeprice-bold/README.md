# @noonnu/wemakeprice-bold
위메프체 Bold - 온라인 쇼핑은 몸도 편하고 마음도 행복하지

## Install
```sh
npm i @noonnu/wemakeprice-bold
```
## Usage
### cjs
```js
require("@noonnu/wemakeprice-bold")
```
### esm
```js
import "@noonnu/wemakeprice-bold"
```
### css-loader
```css
@import "~@noonnu/wemakeprice-bold/index.css"
```
