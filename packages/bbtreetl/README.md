# @noonnu/bbtreetl
유토이미지나무L - 응급실 가기 전에 건강을 챙겨야해요

## Install
```sh
npm i @noonnu/bbtreetl
```
## Usage
### cjs
```js
require("@noonnu/bbtreetl")
```
### esm
```js
import "@noonnu/bbtreetl"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/bbtreetl/index.css"
```
