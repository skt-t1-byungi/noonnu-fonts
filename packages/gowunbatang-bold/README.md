# @noonnu/gowunbatang-bold
고운바탕Bold - 조용한 카페를 좋아한다

## Install
```sh
npm i @noonnu/gowunbatang-bold
```
## Usage
### cjs
```js
require("@noonnu/gowunbatang-bold")
```
### esm
```js
import "@noonnu/gowunbatang-bold"
```
### css-loader
```css
@import "~@noonnu/gowunbatang-bold/index.css"
```
