# @noonnu/coco

코코체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

![example](./example.png)

## Install

```bash
npm install @noonnu/coco --save
```

### Import the CSS file

```js
import '@noonnu/coco' // esm
// or
require('@noonnu/coco') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/coco');
```

## Usage

```css
body {
    font-family: Coco;
}
```
