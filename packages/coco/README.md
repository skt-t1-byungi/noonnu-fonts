# @noonnu/coco
코코체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/coco
```
## Import
### CJS
```js
require("@noonnu/coco")
```
### ESM
```js
import "@noonnu/coco"
```
### CSS 
```css
@import "~@noonnu/coco/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Coco", serif;
}
```
