# @noonnu/gyungyoung
온글잎 경영체 - 마라는 역시 맛있다

## Install
```sh
npm i @noonnu/gyungyoung
```
## Usage
### cjs
```js
require("@noonnu/gyungyoung")
```
### esm
```js
import "@noonnu/gyungyoung"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/gyungyoung/index.css"
```
