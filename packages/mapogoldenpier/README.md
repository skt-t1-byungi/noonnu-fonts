```sh
npm i @noonnu/mapogoldenpier
```

### cjs
```js
require("@noonnu/mapogoldenpier")
```
### esm
```js
import "@noonnu/mapogoldenpier"
```
### css-loader
```css
@import "~@noonnu/mapogoldenpier/index.css"
```
