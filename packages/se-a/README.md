# @noonnu/se-a

세아체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

![example](./example.png)

## Install

```bash
npm install @noonnu/se-a --save
```

### Import the CSS file

```js
import '@noonnu/se-a' // esm
// or
require('@noonnu/se-a') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/se-a');
```

## Usage

```css
body {
    font-family: Se-a;
}
```
