# @noonnu/se-a
세아체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/se-a
```
## Import
### CJS
```js
require("@noonnu/se-a")
```
### ESM
```js
import "@noonnu/se-a"
```
### CSS 
```css
@import "~@noonnu/se-a/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Se-a", serif;
}
```
