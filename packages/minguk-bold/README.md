# @noonnu/minguk-bold
민국체 Bold - 취업은 왜 이렇게 힘드냐

## Install
```sh
npm i @noonnu/minguk-bold
```
## Usage
### cjs
```js
require("@noonnu/minguk-bold")
```
### esm
```js
import "@noonnu/minguk-bold"
```
### css-loader
```css
@import "~@noonnu/minguk-bold/index.css"
```
