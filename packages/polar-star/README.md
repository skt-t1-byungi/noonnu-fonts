# @noonnu/polar-star
북극성 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/polar-star
```
## Usage
### cjs
```js
require("@noonnu/polar-star")
```
### esm
```js
import "@noonnu/polar-star"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/polar-star/index.css"
```
