# @noonnu/busan
부산체 - 부산의 앞바다는 정말 파래서 좋다

## Install
```sh
npm i @noonnu/busan
```
## Import
### CJS
```js
require("@noonnu/busan")
```
### ESM
```js
import "@noonnu/busan"
```
### CSS 
```css
@import "~@noonnu/busan/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Busan", serif;
}
```
