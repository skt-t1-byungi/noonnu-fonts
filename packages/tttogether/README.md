# @noonnu/tttogether
TT투게더 - 디자이너와 함께하는 투게더

## Install
```sh
npm i @noonnu/tttogether
```
## Usage
### cjs
```js
require("@noonnu/tttogether")
```
### esm
```js
import "@noonnu/tttogether"
```
### css-loader
```css
@import "~@noonnu/tttogether/index.css"
```
