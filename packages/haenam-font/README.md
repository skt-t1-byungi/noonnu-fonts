# @noonnu/haenam-font

해남체 - 벌써 날씨가 따뜻해지고 있다

![example](./example.png)

## Install

```bash
npm install @noonnu/haenam-font --save
```

### Import the CSS file

```js
import '@noonnu/haenam-font' // esm
// or
require('@noonnu/haenam-font') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/haenam-font');
```

## Usage

```css
body {
    font-family: HaenamFont;
}
```
