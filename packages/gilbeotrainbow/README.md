# @noonnu/gilbeotrainbow
길벗체 Rainbow - 알록달록한 폰트인데.. 여기서는 색이 안나와..

## Install
```sh
npm i @noonnu/gilbeotrainbow
```
## Usage
### cjs
```js
require("@noonnu/gilbeotrainbow")
```
### esm
```js
import "@noonnu/gilbeotrainbow"
```
### css-loader
```css
@import "~@noonnu/gilbeotrainbow/index.css"
```
