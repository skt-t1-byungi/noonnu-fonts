# @noonnu/nixgonl-vb
닉스곤폰트L - 많이 추운날에 가버리다니 무심하네

## Install
```sh
npm i @noonnu/nixgonl-vb
```
## Usage
### cjs
```js
require("@noonnu/nixgonl-vb")
```
### esm
```js
import "@noonnu/nixgonl-vb"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/nixgonl-vb/index.css"
```
