# @noonnu/gosanja

고산자 - 대동여지도의 위대한 정신을 이어받아

![example](./example.png)

## Install

```bash
npm install @noonnu/gosanja --save
```

### Import the CSS file

```js
import '@noonnu/gosanja' // esm
// or
require('@noonnu/gosanja') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/gosanja');
```

## Usage

```css
body {
    font-family: Gosanja;
}
```
