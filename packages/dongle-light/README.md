# @noonnu/dongle-light
동글Light - 아무것도 안하고 굴러다니고 싶다

## Install
```sh
npm i @noonnu/dongle-light
```
## Usage
### cjs
```js
require("@noonnu/dongle-light")
```
### esm
```js
import "@noonnu/dongle-light"
```
### css-loader
```css
@import "~@noonnu/dongle-light/index.css"
```
