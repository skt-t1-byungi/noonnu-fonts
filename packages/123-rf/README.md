# @noonnu/123-rf

123RF - 곧 봄이라는데 눈이 내리네ㅎㅎㅎ

![example](./example.png)

## Install

```bash
npm install @noonnu/123-rf --save
```

### Import the CSS file

```js
import '@noonnu/123-rf' // esm
// or
require('@noonnu/123-rf') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/123-rf');
```

## Usage

```css
body {
    font-family: 123RF;
}
```
