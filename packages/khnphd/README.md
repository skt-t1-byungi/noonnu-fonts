# @noonnu/khnphd
한수원 한돋움체 - 아침 잘 챙겨먹고 다녀

## Install
```sh
npm i @noonnu/khnphd
```
## Import
### CJS
```js
require("@noonnu/khnphd")
```
### ESM
```js
import "@noonnu/khnphd"
```
### CSS 
```css
@import "~@noonnu/khnphd/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "KHNPHD", serif;
}
```
