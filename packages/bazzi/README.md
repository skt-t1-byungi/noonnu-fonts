# @noonnu/bazzi
넥슨 배찌체 - 한손은 물풍선을 잡고, 한손은 핸들을 잡고

## Install
```sh
npm i @noonnu/bazzi
```
## Import
### CJS
```js
require("@noonnu/bazzi")
```
### ESM
```js
import "@noonnu/bazzi"
```
### CSS 
```css
@import "~@noonnu/bazzi/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Bazzi", serif;
}
```
