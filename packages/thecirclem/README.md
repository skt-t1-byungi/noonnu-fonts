# @noonnu/thecirclem
동그라미체 - 의식의 흐름이 절정에 달하였을 때

## Install
```sh
npm i @noonnu/thecirclem
```
## Usage
### cjs
```js
require("@noonnu/thecirclem")
```
### esm
```js
import "@noonnu/thecirclem"
```
### css-loader
```css
@import "~@noonnu/thecirclem/index.css"
```
