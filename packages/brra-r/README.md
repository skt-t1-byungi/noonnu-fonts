# @noonnu/brra-r
배스킨라빈스R - 이한치한으로 겨울에 아이스크림

## Install
```sh
npm i @noonnu/brra-r
```
## Import
### CJS
```js
require("@noonnu/brra-r")
```
### ESM
```js
import "@noonnu/brra-r"
```
### CSS 
```css
@import "~@noonnu/brra-r/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "BRRA_R", serif;
}
```
