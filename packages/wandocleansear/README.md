# @noonnu/wandocleansear
완도청정바다체 Regular - 깨끗한 바다를 헤엄치는 물고기들

## Install
```sh
npm i @noonnu/wandocleansear
```
## Usage
### cjs
```js
require("@noonnu/wandocleansear")
```
### esm
```js
import "@noonnu/wandocleansear"
```
### css-loader
```css
@import "~@noonnu/wandocleansear/index.css"
```
