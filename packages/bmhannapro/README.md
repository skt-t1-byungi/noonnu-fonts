# @noonnu/bmhannapro
한나체Pro - 치킨이라고 치면 칰이 나온대

## Install
```sh
npm i @noonnu/bmhannapro
```
## Usage
### cjs
```js
require("@noonnu/bmhannapro")
```
### esm
```js
import "@noonnu/bmhannapro"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/bmhannapro/index.css"
```
