# @noonnu/mingijuk
밍기적체 - 밍기적거리면서 유튜브를 보자

## Install
```sh
npm i @noonnu/mingijuk
```
## Usage
### cjs
```js
require("@noonnu/mingijuk")
```
### esm
```js
import "@noonnu/mingijuk"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/mingijuk/index.css"
```
