```sh
npm i @noonnu/oseongandhaneum
```
## Usage
### cjs
```js
require("@noonnu/oseongandhaneum")
```
### esm
```js
import "@noonnu/oseongandhaneum"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/oseongandhaneum/index.css"
```
