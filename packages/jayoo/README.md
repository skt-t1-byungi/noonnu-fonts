# @noonnu/jayoo

아줌마 자유 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

![example](./example.png)

## Install

```bash
npm install @noonnu/jayoo --save
```

### Import the CSS file

```js
import '@noonnu/jayoo' // esm
// or
require('@noonnu/jayoo') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/jayoo');
```

## Usage

```css
body {
    font-family: Jayoo;
}
```
