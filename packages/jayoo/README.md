# @noonnu/jayoo
아줌마 자유 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/jayoo
```
## Import
### CJS
```js
require("@noonnu/jayoo")
```
### ESM
```js
import "@noonnu/jayoo"
```
### CSS 
```css
@import "~@noonnu/jayoo/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Jayoo", serif;
}
```
