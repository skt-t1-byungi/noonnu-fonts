# @noonnu/hyejun

혜준체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

![example](./example.png)

## Install

```bash
npm install @noonnu/hyejun --save
```

### Import the CSS file

```js
import '@noonnu/hyejun' // esm
// or
require('@noonnu/hyejun') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/hyejun');
```

## Usage

```css
body {
    font-family: Hyejun;
}
```
