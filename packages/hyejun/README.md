# @noonnu/hyejun
혜준체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/hyejun
```
## Import
### CJS
```js
require("@noonnu/hyejun")
```
### ESM
```js
import "@noonnu/hyejun"
```
### CSS 
```css
@import "~@noonnu/hyejun/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Hyejun", serif;
}
```
