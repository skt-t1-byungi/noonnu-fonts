# @noonnu/sf-hambaksnow

## Install
```sh
npm i @noonnu/sf-hambaksnow
```
## Usage
### cjs
```js
require("@noonnu/sf-hambaksnow")
```
### esm
```js
import "@noonnu/sf-hambaksnow"
```
### css-loader
```css
@import "~@noonnu/sf-hambaksnow/index.css"
```
