# @noonnu/eoe-ye-b

남양주다산 - 정약용은 조선의 실학자다

![example](./example.png)

## Install

```bash
npm install @noonnu/eoe-ye-b --save
```

### Import the CSS file

```js
import '@noonnu/eoe-ye-b' // esm
// or
require('@noonnu/eoe-ye-b') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/eoe-ye-b');
```

## Usage

```css
body {
    font-family: Eoe_YE_B;
}
```
