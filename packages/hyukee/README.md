# @noonnu/hyukee
혁이체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/hyukee
```
## Import
### CJS
```js
require("@noonnu/hyukee")
```
### ESM
```js
import "@noonnu/hyukee"
```
### CSS 
```css
@import "~@noonnu/hyukee/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Hyukee", serif;
}
```
