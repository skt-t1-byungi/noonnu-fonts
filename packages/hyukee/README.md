# @noonnu/hyukee

혁이체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

![example](./example.png)

## Install

```bash
npm install @noonnu/hyukee --save
```

### Import the CSS file

```js
import '@noonnu/hyukee' // esm
// or
require('@noonnu/hyukee') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/hyukee');
```

## Usage

```css
body {
    font-family: Hyukee;
}
```
