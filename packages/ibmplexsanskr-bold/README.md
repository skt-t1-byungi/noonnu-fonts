# @noonnu/ibmplexsanskr-bold
IBM Plex Sans Bold - IBM이 만든 깔끔한 고딕 폰트

## Install
```sh
npm i @noonnu/ibmplexsanskr-bold
```
## Usage
### cjs
```js
require("@noonnu/ibmplexsanskr-bold")
```
### esm
```js
import "@noonnu/ibmplexsanskr-bold"
```
### css-loader
```css
@import "~@noonnu/ibmplexsanskr-bold/index.css"
```
