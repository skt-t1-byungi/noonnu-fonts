# @noonnu/ef-nuly

널리고딕 - 코로나가 다 끝난것처럼 느껴진다

![example](./example.png)

## Install

```bash
npm install @noonnu/ef-nuly --save
```

### Import the CSS file

```js
import '@noonnu/ef-nuly' // esm
// or
require('@noonnu/ef-nuly') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/ef-nuly');
```

## Usage

```css
body {
    font-family: EF_NULY;
}
```
