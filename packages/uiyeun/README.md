# @noonnu/uiyeun
온글잎 의연체 - 아 여행가고 싶다아아

## Install
```sh
npm i @noonnu/uiyeun
```
## Import
### CJS
```js
require("@noonnu/uiyeun")
```
### ESM
```js
import "@noonnu/uiyeun"
```
### CSS 
```css
@import "~@noonnu/uiyeun/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Uiyeun", serif;
}
```
