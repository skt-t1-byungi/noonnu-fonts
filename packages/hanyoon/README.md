# @noonnu/hanyoon

한윤체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

![example](./example.png)

## Install

```bash
npm install @noonnu/hanyoon --save
```

### Import the CSS file

```js
import '@noonnu/hanyoon' // esm
// or
require('@noonnu/hanyoon') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/hanyoon');
```

## Usage

```css
body {
    font-family: Hanyoon;
}
```
