# @noonnu/hanyoon
한윤체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/hanyoon
```
## Usage
### cjs
```js
require("@noonnu/hanyoon")
```
### esm
```js
import "@noonnu/hanyoon"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/hanyoon/index.css"
```
