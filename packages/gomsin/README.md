# @noonnu/gomsin

곰신체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

![example](./example.png)

## Install

```bash
npm install @noonnu/gomsin --save
```

### Import the CSS file

```js
import '@noonnu/gomsin' // esm
// or
require('@noonnu/gomsin') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/gomsin');
```

## Usage

```css
body {
    font-family: Gomsin;
}
```
