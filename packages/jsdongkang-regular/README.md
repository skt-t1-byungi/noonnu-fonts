# @noonnu/jsdongkang-regular
정선동강체 Regular - 만수산 검은 구름이 막 모여든다

## Install
```sh
npm i @noonnu/jsdongkang-regular
```
## Usage
### cjs
```js
require("@noonnu/jsdongkang-regular")
```
### esm
```js
import "@noonnu/jsdongkang-regular"
```
### css-loader
```css
@import "~@noonnu/jsdongkang-regular/index.css"
```
