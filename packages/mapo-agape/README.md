# @noonnu/mapo-agape

마포애민 - 눈누는 사랑, 그 자체입니다.

![example](./example.png)

## Install

```bash
npm install @noonnu/mapo-agape --save
```

### Import the CSS file

```js
import '@noonnu/mapo-agape' // esm
// or
require('@noonnu/mapo-agape') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/mapo-agape');
```

## Usage

```css
body {
    font-family: MapoAgape;
}
```
