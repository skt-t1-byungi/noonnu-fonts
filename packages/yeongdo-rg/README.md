# @noonnu/yeongdo-rg

영도체 - 오잉또잉 신기한 글씨체

![example](./example.png)

## Install

```bash
npm install @noonnu/yeongdo-rg --save
```

### Import the CSS file

```js
import '@noonnu/yeongdo-rg' // esm
// or
require('@noonnu/yeongdo-rg') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/yeongdo-rg');
```

## Usage

```css
body {
    font-family: Yeongdo-Rg;
}
```
