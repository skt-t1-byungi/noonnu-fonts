# @noonnu/kcc-kimhoon
KCC김훈체 - 눈꽃의 화음에 귀를 적신다

## Install
```sh
npm i @noonnu/kcc-kimhoon
```
## Usage
### cjs
```js
require("@noonnu/kcc-kimhoon")
```
### esm
```js
import "@noonnu/kcc-kimhoon"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/kcc-kimhoon/index.css"
```
