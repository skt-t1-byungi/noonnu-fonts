# @noonnu/somi

소미체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

![example](./example.png)

## Install

```bash
npm install @noonnu/somi --save
```

### Import the CSS file

```js
import '@noonnu/somi' // esm
// or
require('@noonnu/somi') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/somi');
```

## Usage

```css
body {
    font-family: Somi;
}
```
