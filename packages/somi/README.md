# @noonnu/somi
소미체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/somi
```
## Import
### CJS
```js
require("@noonnu/somi")
```
### ESM
```js
import "@noonnu/somi"
```
### CSS 
```css
@import "~@noonnu/somi/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Somi", serif;
}
```
