# @noonnu/sf-icemango
망고빙수 - 망고가 좋아서 망고빙수도 좋아

## Install
```sh
npm i @noonnu/sf-icemango
```
## Usage
### cjs
```js
require("@noonnu/sf-icemango")
```
### esm
```js
import "@noonnu/sf-icemango"
```
### css-loader
```css
@import "~@noonnu/sf-icemango/index.css"
```
