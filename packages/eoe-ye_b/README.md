# @noonnu/eoe-ye_b
남양주다산B - 정약용은 조선의 실학자다

## Install
```sh
npm i @noonnu/eoe-ye_b
```
## Usage
### cjs
```js
require("@noonnu/eoe-ye_b")
```
### esm
```js
import "@noonnu/eoe-ye_b"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/eoe-ye_b/index.css"
```
