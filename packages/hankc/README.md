# @noonnu/hankc

한겨레결체 - 삶의 과정은 선택의 연속이다

![example](./example.png)

## Install

```bash
npm install @noonnu/hankc --save
```

### Import the CSS file

```js
import '@noonnu/hankc' // esm
// or
require('@noonnu/hankc') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/hankc');
```

## Usage

```css
body {
    font-family: Hankc;
}
```
