# @noonnu/123rf
123RF - 곧 봄이라는데 눈이 내리네ㅎㅎㅎ

## Install
```sh
npm i @noonnu/123rf
```
## Import
### CJS
```js
require("@noonnu/123rf")
```
### ESM
```js
import "@noonnu/123rf"
```
### CSS 
```css
@import "~@noonnu/123rf/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "123RF", serif;
}
```
