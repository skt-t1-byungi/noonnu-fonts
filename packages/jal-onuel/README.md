# @noonnu/jal-onuel
잘풀리는오늘체 - 이 폰트를 사용하면 왠지 오늘이 잘 풀릴 것 같다

## Install
```sh
npm i @noonnu/jal-onuel
```
## Usage
### cjs
```js
require("@noonnu/jal-onuel")
```
### esm
```js
import "@noonnu/jal-onuel"
```
### css-loader
```css
@import "~@noonnu/jal-onuel/index.css"
```
