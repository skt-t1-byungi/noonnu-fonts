# @noonnu/bmjua
주아체 - 우아한형제들은 배민을 만들었죠

## Install
```sh
npm i @noonnu/bmjua
```
## Import
### CJS
```js
require("@noonnu/bmjua")
```
### ESM
```js
import "@noonnu/bmjua"
```
### CSS 
```css
@import "~@noonnu/bmjua/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "BMJUA", serif;
}
```
