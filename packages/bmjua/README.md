# @noonnu/bmjua

주아체 - 우아한형제들은 배민을 만들었죠

![example](./example.png)

## Install

```bash
npm install @noonnu/bmjua --save
```

### Import the CSS file

```js
import '@noonnu/bmjua' // esm
// or
require('@noonnu/bmjua') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/bmjua');
```

## Usage

```css
body {
    font-family: BMJUA;
}
```
