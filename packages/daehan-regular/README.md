# @noonnu/daehan-regular
대한체 Regular - 아침에 일어나서 먹는 모닝빵 맛있다

## Install
```sh
npm i @noonnu/daehan-regular
```
## Usage
### cjs
```js
require("@noonnu/daehan-regular")
```
### esm
```js
import "@noonnu/daehan-regular"
```
### css-loader
```css
@import "~@noonnu/daehan-regular/index.css"
```
