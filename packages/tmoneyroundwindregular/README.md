티머니 둥근바람 Regular - 한라봉 랑그드샤 먹고 싶다

## Install
```sh
npm i @noonnu/tmoneyroundwindregular
```

### cjs
```js
require("@noonnu/tmoneyroundwindregular")
```
### esm
```js
import "@noonnu/tmoneyroundwindregular"
```
### css-loader
```css
@import "~@noonnu/tmoneyroundwindregular/index.css"
```
