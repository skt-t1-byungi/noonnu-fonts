```sh
npm i @noonnu/sdkukdetopokki-lt
```

### cjs
```js
require("@noonnu/sdkukdetopokki-lt")
```
### esm
```js
import "@noonnu/sdkukdetopokki-lt"
```
### css-loader

```css
@import "~@noonnu/sdkukdetopokki-lt/index.css"
```
