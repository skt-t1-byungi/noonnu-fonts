```sh
npm i @noonnu/dongle-regular
```

### cjs
```js
require("@noonnu/dongle-regular")
```
### esm
```js
import "@noonnu/dongle-regular"
```
### css-loader
```css
@import "~@noonnu/dongle-regular/index.css"
```
