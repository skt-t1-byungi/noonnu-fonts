# @noonnu/bitgoeul-bold
빛고을광주체Bold - 광주는 대한민국의 서남부에 있는 광역시입니다.

## Install
```sh
npm i @noonnu/bitgoeul-bold
```
## Usage
### cjs
```js
require("@noonnu/bitgoeul-bold")
```
### esm
```js
import "@noonnu/bitgoeul-bold"
```
### css-loader
```css
@import "~@noonnu/bitgoeul-bold/index.css"
```
