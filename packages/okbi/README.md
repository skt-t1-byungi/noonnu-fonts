# @noonnu/okbi

옥비체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

![example](./example.png)

## Install

```bash
npm install @noonnu/okbi --save
```

### Import the CSS file

```js
import '@noonnu/okbi' // esm
// or
require('@noonnu/okbi') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/okbi');
```

## Usage

```css
body {
    font-family: Okbi;
}
```
