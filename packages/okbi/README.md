# @noonnu/okbi
옥비체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/okbi
```
## Usage
### cjs
```js
require("@noonnu/okbi")
```
### esm
```js
import "@noonnu/okbi"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/okbi/index.css"
```
