# @noonnu/yesgothic-regular
예스 고딕 Regular - 책을 꾸준히 읽는 사람이 되자

## Install
```sh
npm i @noonnu/yesgothic-regular
```
## Usage
### cjs
```js
require("@noonnu/yesgothic-regular")
```
### esm
```js
import "@noonnu/yesgothic-regular"
```
### css-loader
```css
@import "~@noonnu/yesgothic-regular/index.css"
```
