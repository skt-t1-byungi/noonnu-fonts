# @noonnu/treegarden
나무정원 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/treegarden
```
## Usage
### cjs
```js
require("@noonnu/treegarden")
```
### esm
```js
import "@noonnu/treegarden"
```
### css-loader
```css
@import "~@noonnu/treegarden/index.css"
```
