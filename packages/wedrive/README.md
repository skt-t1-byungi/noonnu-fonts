# @noonnu/wedrive

위드라이브 전용서체 - 나도 운전을 해보고 싶다

![example](./example.png)

## Install

```bash
npm install @noonnu/wedrive --save
```

### Import the CSS file

```js
import '@noonnu/wedrive' // esm
// or
require('@noonnu/wedrive') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/wedrive');
```

## Usage

```css
body {
    font-family: wedrive;
}
```
