# @noonnu/kopub-batang
KoPub바탕 - 나는 펜의 사각거리는 소리가 참 좋았어

## Install
```sh
npm i @noonnu/kopub-batang
```
## Usage
### cjs
```js
require("@noonnu/kopub-batang")
```
### esm
```js
import "@noonnu/kopub-batang"
```
### css-loader
```css
@import "~@noonnu/kopub-batang/index.css"
```
