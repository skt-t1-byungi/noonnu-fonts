# @noonnu/kotraleap
코트라 도약체 - 굵직한 세리프체

## Install
```sh
npm i @noonnu/kotraleap
```
## Import
### CJS
```js
require("@noonnu/kotraleap")
```
### ESM
```js
import "@noonnu/kotraleap"
```
### CSS 
```css
@import "~@noonnu/kotraleap/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "KOTRALEAP", serif;
}
```
