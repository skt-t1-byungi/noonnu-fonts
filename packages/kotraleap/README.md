# @noonnu/kotraleap

코트라 도약체 - 굵직한 세리프체

![example](./example.png)

## Install

```bash
npm install @noonnu/kotraleap --save
```

### Import the CSS file

```js
import '@noonnu/kotraleap' // esm
// or
require('@noonnu/kotraleap') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/kotraleap');
```

## Usage

```css
body {
    font-family: KOTRALEAP;
}
```
