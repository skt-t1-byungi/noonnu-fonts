# @noonnu/jeju-gothic
제주고딕 - 여행을 많이 다니다 보면 알게 될거야

## Install
```sh
npm i @noonnu/jeju-gothic
```
## Usage
### cjs
```js
require("@noonnu/jeju-gothic")
```
### esm
```js
import "@noonnu/jeju-gothic"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/jeju-gothic/index.css"
```
