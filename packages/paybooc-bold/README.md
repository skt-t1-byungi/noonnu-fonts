# @noonnu/paybooc-bold
페이북 글꼴 Bold - 상큼한 사과에이드 마시고 싶다

## Install
```sh
npm i @noonnu/paybooc-bold
```
## Usage
### cjs
```js
require("@noonnu/paybooc-bold")
```
### esm
```js
import "@noonnu/paybooc-bold"
```
### css-loader
```css
@import "~@noonnu/paybooc-bold/index.css"
```
