# @noonnu/hanna
한나체 - 우아한형제들의 폰트가 적용되어 있습니다.

## Install
```sh
npm i @noonnu/hanna
```
## Import
### CJS
```js
require("@noonnu/hanna")
```
### ESM
```js
import "@noonnu/hanna"
```
### CSS 
```css
@import "~@noonnu/hanna/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Hanna", serif;
}
```
