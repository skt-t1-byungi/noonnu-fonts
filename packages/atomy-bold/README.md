# @noonnu/atomy-bold
애터미체 - 글로벌 인재가 되어서 해외로 나가고 싶다

## Install
```sh
npm i @noonnu/atomy-bold
```
## Usage
### cjs
```js
require("@noonnu/atomy-bold")
```
### esm
```js
import "@noonnu/atomy-bold"
```
### css-loader
```css
@import "~@noonnu/atomy-bold/index.css"
```
