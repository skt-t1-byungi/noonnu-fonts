# @noonnu/rose
장미체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/rose
```
## Import
### CJS
```js
require("@noonnu/rose")
```
### ESM
```js
import "@noonnu/rose"
```
### CSS 
```css
@import "~@noonnu/rose/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Rose", serif;
}
```
