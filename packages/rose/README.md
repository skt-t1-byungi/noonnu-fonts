# @noonnu/rose

장미체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

![example](./example.png)

## Install

```bash
npm install @noonnu/rose --save
```

### Import the CSS file

```js
import '@noonnu/rose' // esm
// or
require('@noonnu/rose') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/rose');
```

## Usage

```css
body {
    font-family: Rose;
}
```
