# @noonnu/wandohopeb
완도희망체 Bold - 바닷가에 다녀 오면 추억이 남지

## Install
```sh
npm i @noonnu/wandohopeb
```
## Usage
### cjs
```js
require("@noonnu/wandohopeb")
```
### esm
```js
import "@noonnu/wandohopeb"
```
### css-loader
```css
@import "~@noonnu/wandohopeb/index.css"
```
