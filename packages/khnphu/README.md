# @noonnu/khnphu
한수원 한울림체 - 당신의 창에 반딧불을 보낼게요

## Install
```sh
npm i @noonnu/khnphu
```
## Import
### CJS
```js
require("@noonnu/khnphu")
```
### ESM
```js
import "@noonnu/khnphu"
```
### CSS 
```css
@import "~@noonnu/khnphu/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "KHNPHU", serif;
}
```
