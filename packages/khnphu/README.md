# @noonnu/khnphu

한수원 한울림체 - 당신의 창에 반딧불을 보낼게요

![example](./example.png)

## Install

```bash
npm install @noonnu/khnphu --save
```

### Import the CSS file

```js
import '@noonnu/khnphu' // esm
// or
require('@noonnu/khnphu') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/khnphu');
```

## Usage

```css
body {
    font-family: KHNPHU;
}
```
