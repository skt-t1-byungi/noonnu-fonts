# @noonnu/hssummer
HS여름물빛체 - 무더운 밤, 잠은 오지 않고

## Install
```sh
npm i @noonnu/hssummer
```
## Usage
### cjs
```js
require("@noonnu/hssummer")
```
### esm
```js
import "@noonnu/hssummer"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/hssummer/index.css"
```
