# @noonnu/gilbeot-bi

길벗체 BI - 주말에 회사일 하지말자

![example](./example.png)

## Install

```bash
npm install @noonnu/gilbeot-bi --save
```

### Import the CSS file

```js
import '@noonnu/gilbeot-bi' // esm
// or
require('@noonnu/gilbeot-bi') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/gilbeot-bi');
```

## Usage

```css
body {
    font-family: GilbeotBI;
}
```
