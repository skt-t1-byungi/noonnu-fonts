# @noonnu/yoon-sjy

Yoon 지영 - 사람마다 손글씨가 다 다르다

![example](./example.png)

## Install

```bash
npm install @noonnu/yoon-sjy --save
```

### Import the CSS file

```js
import '@noonnu/yoon-sjy' // esm
// or
require('@noonnu/yoon-sjy') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/yoon-sjy');
```

## Usage

```css
body {
    font-family: YoonSJY;
}
```
