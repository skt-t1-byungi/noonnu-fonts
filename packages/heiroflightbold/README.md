빛의계승자체 Bold - 날카로운 게임에 잘 어울리는 세리프 폰트

## Install
```sh
npm i @noonnu/heiroflightbold
```
## Usage
### cjs
```js
require("@noonnu/heiroflightbold")
```
### esm
```js
import "@noonnu/heiroflightbold"
```
### css-loader
```css
@import "~@noonnu/heiroflightbold/index.css"
```
