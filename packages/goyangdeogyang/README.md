# @noonnu/goyangdeogyang
고양덕양체 - 덕양은 고양의 다른 이름이다

## Install
```sh
npm i @noonnu/goyangdeogyang
```
## Usage
### cjs
```js
require("@noonnu/goyangdeogyang")
```
### esm
```js
import "@noonnu/goyangdeogyang"
```
### css-loader
```css
@import "~@noonnu/goyangdeogyang/index.css"
```
