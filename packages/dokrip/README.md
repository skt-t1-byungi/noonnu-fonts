# @noonnu/dokrip
독립체 - 대한민국 독립을 기념하는 폰트

## Install
```sh
npm i @noonnu/dokrip
```
## Usage
### cjs
```js
require("@noonnu/dokrip")
```
### esm
```js
import "@noonnu/dokrip"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/dokrip/index.css"
```
