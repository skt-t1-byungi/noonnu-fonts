# @noonnu/dokrip
독립체 - 대한민국 독립을 기념하는 폰트

## Install
```sh
npm i @noonnu/dokrip
```
## Import
### CJS
```js
require("@noonnu/dokrip")
```
### ESM
```js
import "@noonnu/dokrip"
```
### CSS 
```css
@import "~@noonnu/dokrip/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Dokrip", serif;
}
```
