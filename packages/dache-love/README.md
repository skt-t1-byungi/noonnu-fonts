# @noonnu/dache-love
다채사랑 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/dache-love
```
## Usage
### cjs
```js
require("@noonnu/dache-love")
```
### esm
```js
import "@noonnu/dache-love"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/dache-love/index.css"
```
