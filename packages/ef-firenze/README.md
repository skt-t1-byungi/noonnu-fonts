# @noonnu/ef-firenze

피렌체 - 피렌체는 한번도 안가봤다

![example](./example.png)

## Install

```bash
npm install @noonnu/ef-firenze --save
```

### Import the CSS file

```js
import '@noonnu/ef-firenze' // esm
// or
require('@noonnu/ef-firenze') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/ef-firenze');
```

## Usage

```css
body {
    font-family: EF_Firenze;
}
```
