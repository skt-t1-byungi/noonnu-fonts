# @noonnu/handletter
손편지체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/handletter
```
## Usage
### cjs
```js
require("@noonnu/handletter")
```
### esm
```js
import "@noonnu/handletter"
```
### css-loader
```css
@import "~@noonnu/handletter/index.css"
```
