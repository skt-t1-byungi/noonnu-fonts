# @noonnu/mylottelight

## Install
```sh
npm i @noonnu/mylottelight
```
## Usage
### cjs
```js
require("@noonnu/mylottelight")
```
### esm
```js
import "@noonnu/mylottelight"
```
### css-loader
```css
@import "~@noonnu/mylottelight/index.css"
```
