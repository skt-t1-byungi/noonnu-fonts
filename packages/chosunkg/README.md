# @noonnu/chosunkg
조선굵은고딕 - 시간이 참 빠르다

## Install
```sh
npm i @noonnu/chosunkg
```
## Import
### CJS
```js
require("@noonnu/chosunkg")
```
### ESM
```js
import "@noonnu/chosunkg"
```
### CSS 
```css
@import "~@noonnu/chosunkg/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "ChosunKg", serif;
}
```
