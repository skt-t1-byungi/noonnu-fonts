# @noonnu/yes24
예스체 - 다양한 문화 콘텐츠가 뻗어나가는 공간

## Install
```sh
npm i @noonnu/yes24
```
## Import
### CJS
```js
require("@noonnu/yes24")
```
### ESM
```js
import "@noonnu/yes24"
```
### CSS 
```css
@import "~@noonnu/yes24/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "YES24", serif;
}
```
