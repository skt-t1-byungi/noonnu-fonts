# @noonnu/yes24

예스체 - 다양한 문화 콘텐츠가 뻗어나가는 공간

![example](./example.png)

## Install

```bash
npm install @noonnu/yes24 --save
```

### Import the CSS file

```js
import '@noonnu/yes24' // esm
// or
require('@noonnu/yes24') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/yes24');
```

## Usage

```css
body {
    font-family: YES24;
}
```
