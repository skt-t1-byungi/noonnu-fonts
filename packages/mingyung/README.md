# @noonnu/mingyung
예쁜 민경체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/mingyung
```
## Usage
### cjs
```js
require("@noonnu/mingyung")
```
### esm
```js
import "@noonnu/mingyung"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/mingyung/index.css"
```
