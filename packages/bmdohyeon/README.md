# @noonnu/bmdohyeon
도현체 - 치맥보단 피맥이 정말 진리지!

## Install
```sh
npm i @noonnu/bmdohyeon
```
## Import
### CJS
```js
require("@noonnu/bmdohyeon")
```
### ESM
```js
import "@noonnu/bmdohyeon"
```
### CSS 
```css
@import "~@noonnu/bmdohyeon/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "BMDOHYEON", serif;
}
```
