# @noonnu/bmdohyeon

도현체 - 치맥보단 피맥이 정말 진리지!

![example](./example.png)

## Install

```bash
npm install @noonnu/bmdohyeon --save
```

### Import the CSS file

```js
import '@noonnu/bmdohyeon' // esm
// or
require('@noonnu/bmdohyeon') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/bmdohyeon');
```

## Usage

```css
body {
    font-family: BMDOHYEON;
}
```
