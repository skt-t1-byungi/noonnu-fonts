# @noonnu/gimje

김제시체 - 눈누 오래오래 할 수 있었으면 좋겠다

![example](./example.png)

## Install

```bash
npm install @noonnu/gimje --save
```

### Import the CSS file

```js
import '@noonnu/gimje' // esm
// or
require('@noonnu/gimje') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/gimje');
```

## Usage

```css
body {
    font-family: GIMJE;
}
```
