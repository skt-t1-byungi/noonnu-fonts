# @noonnu/mapoagape
마포애민 - 눈누는 사랑, 그 자체입니다.

## Install
```sh
npm i @noonnu/mapoagape
```
## Usage
### cjs
```js
require("@noonnu/mapoagape")
```
### esm
```js
import "@noonnu/mapoagape"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/mapoagape/index.css"
```
