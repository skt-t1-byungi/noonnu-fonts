# @noonnu/mapoagape
마포애민 - 눈누는 사랑, 그 자체입니다.

## Install
```sh
npm i @noonnu/mapoagape
```
## Import
### CJS
```js
require("@noonnu/mapoagape")
```
### ESM
```js
import "@noonnu/mapoagape"
```
### CSS 
```css
@import "~@noonnu/mapoagape/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "MapoAgape", serif;
}
```
