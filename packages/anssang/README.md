# @noonnu/anssang

안쌍체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

![example](./example.png)

## Install

```bash
npm install @noonnu/anssang --save
```

### Import the CSS file

```js
import '@noonnu/anssang' // esm
// or
require('@noonnu/anssang') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/anssang');
```

## Usage

```css
body {
    font-family: Anssang;
}
```
