# @noonnu/sukyung

온글잎 석영체 - 기분전환에는 역시 치킨이다

![example](./example.png)

## Install

```bash
npm install @noonnu/sukyung --save
```

### Import the CSS file

```js
import '@noonnu/sukyung' // esm
// or
require('@noonnu/sukyung') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/sukyung');
```

## Usage

```css
body {
    font-family: Sukyung;
}
```
