# @noonnu/dovemayo-medium

둘기마요 - 둘기마요는 좀 그렇고 치킨마요 먹고 싶다

![example](./example.png)

## Install

```bash
npm install @noonnu/dovemayo-medium --save
```

### Import the CSS file

```js
import '@noonnu/dovemayo-medium' // esm
// or
require('@noonnu/dovemayo-medium') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/dovemayo-medium');
```

## Usage

```css
body {
    font-family: Dovemayo-Medium;
}
```
