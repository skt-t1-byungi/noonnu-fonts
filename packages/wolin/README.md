# @noonnu/wolin
월인석보체 - 옛말 맞는 거 하나도 없더라

## Install
```sh
npm i @noonnu/wolin
```
## Import
### CJS
```js
require("@noonnu/wolin")
```
### ESM
```js
import "@noonnu/wolin"
```
### CSS 
```css
@import "~@noonnu/wolin/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Wolin", serif;
}
```
