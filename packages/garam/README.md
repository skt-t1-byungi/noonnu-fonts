# @noonnu/garam

가람연꽃 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

![example](./example.png)

## Install

```bash
npm install @noonnu/garam --save
```

### Import the CSS file

```js
import '@noonnu/garam' // esm
// or
require('@noonnu/garam') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/garam');
```

## Usage

```css
body {
    font-family: Garam;
}
```
