# @noonnu/garam
가람연꽃 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/garam
```
## Usage
### cjs
```js
require("@noonnu/garam")
```
### esm
```js
import "@noonnu/garam"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/garam/index.css"
```
