# @noonnu/garam
가람연꽃 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/garam
```
## Import
### CJS
```js
require("@noonnu/garam")
```
### ESM
```js
import "@noonnu/garam"
```
### CSS 
```css
@import "~@noonnu/garam/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Garam", serif;
}
```
