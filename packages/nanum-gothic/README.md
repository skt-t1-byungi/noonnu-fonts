# @noonnu/nanum-gothic
나눔고딕 - 파리의 숨은 고양이 찾기

## Install
```sh
npm i @noonnu/nanum-gothic
```
## Usage
### cjs
```js
require("@noonnu/nanum-gothic")
```
### esm
```js
import "@noonnu/nanum-gothic"
```
### css-loader
```css
@import "~@noonnu/nanum-gothic/index.css"
```
