# @noonnu/cute-siu
시우 귀여워 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/cute-siu
```
## Usage
### cjs
```js
require("@noonnu/cute-siu")
```
### esm
```js
import "@noonnu/cute-siu"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/cute-siu/index.css"
```
