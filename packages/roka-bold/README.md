# @noonnu/roka-bold

강한육군 - 우리나라 육군은 강하다

![example](./example.png)

## Install

```bash
npm install @noonnu/roka-bold --save
```

### Import the CSS file

```js
import '@noonnu/roka-bold' // esm
// or
require('@noonnu/roka-bold') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/roka-bold');
```

## Usage

```css
body {
    font-family: ROKABold;
}
```
