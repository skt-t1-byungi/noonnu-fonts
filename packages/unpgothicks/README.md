# @noonnu/unpgothicks
유앤피플 고딕체 - 특별한 사람들이 함께하는 곳

## Install
```sh
npm i @noonnu/unpgothicks
```
## Usage
### cjs
```js
require("@noonnu/unpgothicks")
```
### esm
```js
import "@noonnu/unpgothicks"
```
### css-loader
```css
@import "~@noonnu/unpgothicks/index.css"
```
