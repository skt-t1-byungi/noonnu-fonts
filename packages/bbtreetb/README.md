# @noonnu/bbtreetb
유토이미지나무B - 에그*랍 진짜 맛있어

## Install
```sh
npm i @noonnu/bbtreetb
```
## Usage
### cjs
```js
require("@noonnu/bbtreetb")
```
### esm
```js
import "@noonnu/bbtreetb"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/bbtreetb/index.css"
```
