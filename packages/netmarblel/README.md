# @noonnu/netmarblel
넷마블체L - 크크를 형상화한 귀여운 넷마블체

## Install
```sh
npm i @noonnu/netmarblel
```
## Usage
### cjs
```js
require("@noonnu/netmarblel")
```
### esm
```js
import "@noonnu/netmarblel"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/netmarblel/index.css"
```
