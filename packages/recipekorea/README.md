## Install
```sh
npm i @noonnu/recipekorea
```
## Usage
### cjs
```js
require("@noonnu/recipekorea")
```
### esm
```js
import "@noonnu/recipekorea"
```
### css-loader
```css
@import "~@noonnu/recipekorea/index.css"
```
