# @noonnu/godo
고도체 - 인생 별거 있는 줄 아는데 사실은 별거 없어

## Install
```sh
npm i @noonnu/godo
```
## Import
### CJS
```js
require("@noonnu/godo")
```
### ESM
```js
import "@noonnu/godo"
```
### CSS 
```css
@import "~@noonnu/godo/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Godo", serif;
}
```
