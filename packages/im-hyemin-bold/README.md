# @noonnu/im-hyemin-bold
IM혜민체 Bold - 동글동글 모서리 금융을 더 말랑하게

## Install
```sh
npm i @noonnu/im-hyemin-bold
```
## Usage
### cjs
```js
require("@noonnu/im-hyemin-bold")
```
### esm
```js
import "@noonnu/im-hyemin-bold"
```
### css-loader
```css
@import "~@noonnu/im-hyemin-bold/index.css"
```
