# @noonnu/sungsil

성실체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

![example](./example.png)

## Install

```bash
npm install @noonnu/sungsil --save
```

### Import the CSS file

```js
import '@noonnu/sungsil' // esm
// or
require('@noonnu/sungsil') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/sungsil');
```

## Usage

```css
body {
    font-family: Sungsil;
}
```
