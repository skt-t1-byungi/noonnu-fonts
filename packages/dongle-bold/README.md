# @noonnu/dongle-bold
동글Bold - 동글동글한 동글체

## Install
```sh
npm i @noonnu/dongle-bold
```
## Usage
### cjs
```js
require("@noonnu/dongle-bold")
```
### esm
```js
import "@noonnu/dongle-bold"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/dongle-bold/index.css"
```
