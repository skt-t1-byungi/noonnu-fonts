비트로 프라이드체 - 간결하고 명확한 비트로 프라이드체

## Install
```sh
npm i @noonnu/vitro-pride
```
## Usage
### cjs
```js
require("@noonnu/vitro-pride")
```
### esm
```js
import "@noonnu/vitro-pride"
```
### css-loader
```css
@import "~@noonnu/vitro-pride/index.css"
```
