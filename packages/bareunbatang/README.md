## Install
```sh
npm i @noonnu/bareunbatang
```
## Usage
### cjs
```js
require("@noonnu/bareunbatang")
```
### esm
```js
import "@noonnu/bareunbatang"
```
### css-loader
```css
@import "~@noonnu/bareunbatang/index.css"
```
