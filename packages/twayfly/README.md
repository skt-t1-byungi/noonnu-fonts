# @noonnu/twayfly

티웨이날다체 - 코로나 끝나고 여행가고 싶다

![example](./example.png)

## Install

```bash
npm install @noonnu/twayfly --save
```

### Import the CSS file

```js
import '@noonnu/twayfly' // esm
// or
require('@noonnu/twayfly') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/twayfly');
```

## Usage

```css
body {
    font-family: twayfly;
}
```
