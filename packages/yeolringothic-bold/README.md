# @noonnu/yeolringothic-bold
열린고딕 Bold - 군고구마는 호박고구마가 맛있어

## Install
```sh
npm i @noonnu/yeolringothic-bold
```
## Usage
### cjs
```js
require("@noonnu/yeolringothic-bold")
```
### esm
```js
import "@noonnu/yeolringothic-bold"
```
### css-loader
```css
@import "~@noonnu/yeolringothic-bold/index.css"
```
