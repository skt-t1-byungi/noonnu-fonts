# @noonnu/bohyun
온글잎 보현체 - 얼그레이스콘은 달달

## Install
```sh
npm i @noonnu/bohyun
```
## Import
### CJS
```js
require("@noonnu/bohyun")
```
### ESM
```js
import "@noonnu/bohyun"
```
### CSS 
```css
@import "~@noonnu/bohyun/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Bohyun", serif;
}
```
