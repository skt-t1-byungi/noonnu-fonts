# @noonnu/bohyun

온글잎 보현체 - 얼그레이스콘은 달달

![example](./example.png)

## Install

```bash
npm install @noonnu/bohyun --save
```

### Import the CSS file

```js
import '@noonnu/bohyun' // esm
// or
require('@noonnu/bohyun') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/bohyun');
```

## Usage

```css
body {
    font-family: Bohyun;
}
```
