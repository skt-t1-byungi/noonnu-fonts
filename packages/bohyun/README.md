# @noonnu/bohyun
온글잎 보현체 - 얼그레이스콘은 달달

## Install
```sh
npm i @noonnu/bohyun
```
## Usage
### cjs
```js
require("@noonnu/bohyun")
```
### esm
```js
import "@noonnu/bohyun"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/bohyun/index.css"
```
