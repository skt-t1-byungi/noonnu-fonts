# @noonnu/restart

다시 시작해 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

![example](./example.png)

## Install

```bash
npm install @noonnu/restart --save
```

### Import the CSS file

```js
import '@noonnu/restart' // esm
// or
require('@noonnu/restart') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/restart');
```

## Usage

```css
body {
    font-family: Restart;
}
```
