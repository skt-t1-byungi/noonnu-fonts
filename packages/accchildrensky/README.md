```sh
npm i @noonnu/accchildrensky
```
## Usage
### cjs
```js
require("@noonnu/accchildrensky")
```
### esm
```js
import "@noonnu/accchildrensky"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/accchildrensky/index.css"
```
