# @noonnu/love-son
사랑해 아들 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/love-son
```
## Usage
### cjs
```js
require("@noonnu/love-son")
```
### esm
```js
import "@noonnu/love-son"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/love-son/index.css"
```
