# @noonnu/hope-nuri
희망누리 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/hope-nuri
```
## Usage
### cjs
```js
require("@noonnu/hope-nuri")
```
### esm
```js
import "@noonnu/hope-nuri"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/hope-nuri/index.css"
```
