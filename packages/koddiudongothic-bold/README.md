```sh
npm i @noonnu/koddiudongothic-bold
```
## Usage
### cjs
```js
require("@noonnu/koddiudongothic-bold")
```
### esm
```js
import "@noonnu/koddiudongothic-bold"
```
### css-loader
```css
@import "~@noonnu/koddiudongothic-bold/index.css"
```
