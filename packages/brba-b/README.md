# @noonnu/brba-b

배스킨라빈스 - 빵케익만 먹지말고 아이스크림 케익

![example](./example.png)

## Install

```bash
npm install @noonnu/brba-b --save
```

### Import the CSS file

```js
import '@noonnu/brba-b' // esm
// or
require('@noonnu/brba-b') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/brba-b');
```

## Usage

```css
body {
    font-family: BRBA_B;
}
```
