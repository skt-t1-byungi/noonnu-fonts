# @noonnu/brba-b
배스킨라빈스B - 빵케익만 먹지말고 아이스크림 케익

## Install
```sh
npm i @noonnu/brba-b
```
## Import
### CJS
```js
require("@noonnu/brba-b")
```
### ESM
```js
import "@noonnu/brba-b"
```
### CSS 
```css
@import "~@noonnu/brba-b/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "BRBA_B", serif;
}
```
