# @noonnu/chosun-kg

조선굵은고딕 - 시간이 참 빠르다

![example](./example.png)

## Install

```bash
npm install @noonnu/chosun-kg --save
```

### Import the CSS file

```js
import '@noonnu/chosun-kg' // esm
// or
require('@noonnu/chosun-kg') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/chosun-kg');
```

## Usage

```css
body {
    font-family: ChosunKg;
}
```
