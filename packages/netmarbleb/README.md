# @noonnu/netmarbleb
넷마블체B - 넷마블 캐릭터 크크 귀엽더라

## Install
```sh
npm i @noonnu/netmarbleb
```
## Usage
### cjs
```js
require("@noonnu/netmarbleb")
```
### esm
```js
import "@noonnu/netmarbleb"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/netmarbleb/index.css"
```
