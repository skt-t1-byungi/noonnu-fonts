# @noonnu/b-ccard-b

비씨카드 글꼴 - 혜택 많은 카드로 잘 선택하세요

![example](./example.png)

## Install

```bash
npm install @noonnu/b-ccard-b --save
```

### Import the CSS file

```js
import '@noonnu/b-ccard-b' // esm
// or
require('@noonnu/b-ccard-b') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/b-ccard-b');
```

## Usage

```css
body {
    font-family: BCcardB;
}
```
