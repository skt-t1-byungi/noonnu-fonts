# @noonnu/wemakeprice-regular
위메프체 Regular - 위메프에서 쇼핑을 합시다

## Install
```sh
npm i @noonnu/wemakeprice-regular
```
## Usage
### cjs
```js
require("@noonnu/wemakeprice-regular")
```
### esm
```js
import "@noonnu/wemakeprice-regular"
```
### css-loader
```css
@import "~@noonnu/wemakeprice-regular/index.css"
```
