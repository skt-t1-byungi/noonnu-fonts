```sh
npm i @noonnu/onsaryakunheab
```
## Usage
### cjs
```js
require("@noonnu/onsaryakunheab")
```
### esm
```js
import "@noonnu/onsaryakunheab"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/onsaryakunheab/index.css"
```
