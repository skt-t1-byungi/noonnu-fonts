# @noonnu/vitro-core
비트로 코어체 - 스포티한 느낌의 비트로 코어체

## Install
```sh
npm i @noonnu/vitro-core
```
## Usage
### cjs
```js
require("@noonnu/vitro-core")
```
### esm
```js
import "@noonnu/vitro-core"
```
### css-loader
```css
@import "~@noonnu/vitro-core/index.css"
```
