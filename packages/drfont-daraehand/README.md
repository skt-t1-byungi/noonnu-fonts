# @noonnu/drfont-daraehand
다래손글씨체 - 아름다운 한글을 널리 퍼뜨리자

## Install
```sh
npm i @noonnu/drfont-daraehand
```
## Usage
### cjs
```js
require("@noonnu/drfont-daraehand")
```
### esm
```js
import "@noonnu/drfont-daraehand"
```
### css-loader
```css
@import "~@noonnu/drfont-daraehand/index.css"
```
