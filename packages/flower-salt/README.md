# @noonnu/flower-salt

꽃소금체 - 동글동글 귀여운 손글씨

![example](./example.png)

## Install

```bash
npm install @noonnu/flower-salt --save
```

### Import the CSS file

```js
import '@noonnu/flower-salt' // esm
// or
require('@noonnu/flower-salt') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/flower-salt');
```

## Usage

```css
body {
    font-family: FlowerSalt;
}
```
