# @noonnu/yedang
예당체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/yedang
```
## Import
### CJS
```js
require("@noonnu/yedang")
```
### ESM
```js
import "@noonnu/yedang"
```
### CSS 
```css
@import "~@noonnu/yedang/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Yedang", serif;
}
```
