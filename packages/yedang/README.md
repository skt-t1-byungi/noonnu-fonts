# @noonnu/yedang

예당체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

![example](./example.png)

## Install

```bash
npm install @noonnu/yedang --save
```

### Import the CSS file

```js
import '@noonnu/yedang' // esm
// or
require('@noonnu/yedang') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/yedang');
```

## Usage

```css
body {
    font-family: Yedang;
}
```
