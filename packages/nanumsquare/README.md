```sh
npm i @noonnu/nanumsquare
```

### cjs
```js
require("@noonnu/nanumsquare")
```
### esm
```js
import "@noonnu/nanumsquare"
```
### css-loader
```css
@import "~@noonnu/nanumsquare/index.css"
```
