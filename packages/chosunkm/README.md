# @noonnu/chosunkm
조선굵은명조 - 아무것도 안하고 싶다

## Install
```sh
npm i @noonnu/chosunkm
```
## Import
### CJS
```js
require("@noonnu/chosunkm")
```
### ESM
```js
import "@noonnu/chosunkm"
```
### CSS 
```css
@import "~@noonnu/chosunkm/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "ChosunKm", serif;
}
```
