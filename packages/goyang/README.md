# @noonnu/goyang
고양체 - 고양시의 상징은 고양이다냥

## Install
```sh
npm i @noonnu/goyang
```
## Usage
### cjs
```js
require("@noonnu/goyang")
```
### esm
```js
import "@noonnu/goyang"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/goyang/index.css"
```
