# @noonnu/goyang
고양체 - 고양시의 상징은 고양이다냥

## Install
```sh
npm i @noonnu/goyang
```
## Import
### CJS
```js
require("@noonnu/goyang")
```
### ESM
```js
import "@noonnu/goyang"
```
### CSS 
```css
@import "~@noonnu/goyang/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Goyang", serif;
}
```
