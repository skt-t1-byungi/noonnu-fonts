# @noonnu/gonggothiclight
이사만루 Light - 야구의 세계는 끝이 없다

## Install
```sh
npm i @noonnu/gonggothiclight
```
## Usage
### cjs
```js
require("@noonnu/gonggothiclight")
```
### esm
```js
import "@noonnu/gonggothiclight"
```
### css-loader
```css
@import "~@noonnu/gonggothiclight/index.css"
```
