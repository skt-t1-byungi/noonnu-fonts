# @noonnu/paybooc-extrabold
페이북글꼴 ExtraBold - 제주도에 가면 한라봉 랑그드샤

## Install
```sh
npm i @noonnu/paybooc-extrabold
```
## Usage
### cjs
```js
require("@noonnu/paybooc-extrabold")
```
### esm
```js
import "@noonnu/paybooc-extrabold"
```
### css-loader
```css
@import "~@noonnu/paybooc-extrabold/index.css"
```
