문화재돌봄체 Regular - 문화재청은 문화재를 돌봅니다

## Install
```sh
npm i @noonnu/dolbomche-r
```
## Usage
### cjs
```js
require("@noonnu/dolbomche-r")
```
### esm
```js
import "@noonnu/dolbomche-r"
```
### css-loader
```css
@import "~@noonnu/dolbomche-r/index.css"
```
