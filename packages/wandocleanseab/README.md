# @noonnu/wandocleanseab
완도청정바다체 Bold - 완도에는 장보고 유적지가 유명하다고 해

## Install
```sh
npm i @noonnu/wandocleanseab
```
## Usage
### cjs
```js
require("@noonnu/wandocleanseab")
```
### esm
```js
import "@noonnu/wandocleanseab"
```
### css-loader
```css
@import "~@noonnu/wandocleanseab/index.css"
```
