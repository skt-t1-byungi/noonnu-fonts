# @noonnu/one-mobile-pop
원스토어 모바일POP체 - 튀긴음식만 먹어도 건강했으면

## Install
```sh
npm i @noonnu/one-mobile-pop
```
## Usage
### cjs
```js
require("@noonnu/one-mobile-pop")
```
### esm
```js
import "@noonnu/one-mobile-pop"
```
### css-loader
```css
@import "~@noonnu/one-mobile-pop/index.css"
```
