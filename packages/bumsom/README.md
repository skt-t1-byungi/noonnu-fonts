# @noonnu/bumsom
범솜체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/bumsom
```
## Usage
### cjs
```js
require("@noonnu/bumsom")
```
### esm
```js
import "@noonnu/bumsom"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/bumsom/index.css"
```
