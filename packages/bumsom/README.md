# @noonnu/bumsom
범솜체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/bumsom
```
## Import
### CJS
```js
require("@noonnu/bumsom")
```
### ESM
```js
import "@noonnu/bumsom"
```
### CSS 
```css
@import "~@noonnu/bumsom/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Bumsom", serif;
}
```
