# @noonnu/bumsom

범솜체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

![example](./example.png)

## Install

```bash
npm install @noonnu/bumsom --save
```

### Import the CSS file

```js
import '@noonnu/bumsom' // esm
// or
require('@noonnu/bumsom') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/bumsom');
```

## Usage

```css
body {
    font-family: Bumsom;
}
```
