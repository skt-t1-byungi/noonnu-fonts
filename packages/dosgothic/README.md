# @noonnu/dosgothic
도스고딕 - 뭔가 팩맨이라도 해야할것 같아

## Install
```sh
npm i @noonnu/dosgothic
```
## Usage
### cjs
```js
require("@noonnu/dosgothic")
```
### esm
```js
import "@noonnu/dosgothic"
```
### css (with [css-loader](https://github.com/webpack-contrib/css-loader))
```css
@import "~@noonnu/dosgothic/index.css"
```
