# @noonnu/se-hwa

세화체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

![example](./example.png)

## Install

```bash
npm install @noonnu/se-hwa --save
```

### Import the CSS file

```js
import '@noonnu/se-hwa' // esm
// or
require('@noonnu/se-hwa') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/se-hwa');
```

## Usage

```css
body {
    font-family: Se-hwa;
}
```
