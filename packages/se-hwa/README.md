# @noonnu/se-hwa
세화체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/se-hwa
```
## Import
### CJS
```js
require("@noonnu/se-hwa")
```
### ESM
```js
import "@noonnu/se-hwa"
```
### CSS 
```css
@import "~@noonnu/se-hwa/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Se-hwa", serif;
}
```
