# @noonnu/mago

마고체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

![example](./example.png)

## Install

```bash
npm install @noonnu/mago --save
```

### Import the CSS file

```js
import '@noonnu/mago' // esm
// or
require('@noonnu/mago') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/mago');
```

## Usage

```css
body {
    font-family: Mago;
}
```
