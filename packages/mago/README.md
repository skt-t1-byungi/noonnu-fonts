# @noonnu/mago
마고체 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

## Install
```sh
npm i @noonnu/mago
```
## Import
### CJS
```js
require("@noonnu/mago")
```
### ESM
```js
import "@noonnu/mago"
```
### CSS 
```css
@import "~@noonnu/mago/index.css"
```
> [css-loader](https://github.com/webpack-contrib/css-loader) are required.

## Usage
```css
body {
    font-family: "Mago", serif;
}
```
