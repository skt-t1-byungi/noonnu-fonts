# @noonnu/ydestreetb
Y이드스트릿체 Bold - 개성넘치는 와이드 폰트

## Install
```sh
npm i @noonnu/ydestreetb
```
## Usage
### cjs
```js
require("@noonnu/ydestreetb")
```
### esm
```js
import "@noonnu/ydestreetb"
```
### css-loader
```css
@import "~@noonnu/ydestreetb/index.css"
```
