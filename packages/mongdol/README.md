# @noonnu/mongdol

몽돌 - 네이버 클로바 AI가 만든 나눔손글씨 글꼴

![example](./example.png)

## Install

```bash
npm install @noonnu/mongdol --save
```

### Import the CSS file

```js
import '@noonnu/mongdol' // esm
// or
require('@noonnu/mongdol') // cjs
```

#### [css-loader](https://github.com/webpack-contrib/css-loader)

```css
@import url('~@noonnu/mongdol');
```

## Usage

```css
body {
    font-family: Mongdol;
}
```
