# Problem & Goal

## Current Pains
- **의존성**: 눈누 데이터 변경 시 파이프라인 중단.
- **일관성 부족**: 동일 패밀리가 분리되거나, 다른 폰트가 섞임.
- **배포 단위 모호**: 패키지 1개가 패밀리인지 웨이트인지 불명확.
- **무거운 레포**: 바이너리 직접 커밋으로 인한 용량/히스토리 비대화.

## v2 Goal
"눈누 종속성 제거" 및 **"사람 결정 기반의 정본(SoT) 구축"**.
- **Human-in-the-loop**: 모호한 케이스는 사람이 결정(LLM 보조).
- **Stable**: 결정된 매니페스트 기반의 재현 가능한 배포.
- **Lightweight**: git엔 메타데이터만, 바이너리는 외부 참조.

## Non-goals
- 100% 완전 자동화 (사람 개입 인정).
- 화려한 UI (CLI/JSON 우선).
